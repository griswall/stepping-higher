# Stepping Higher

Website for Anthony Harrison's coaching practice, Stepping Higher.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Local Development

```bash
npm install
npm run dev
```

The local site runs at `http://127.0.0.1:5173/` by default.

## Production Build

```bash
npm run build
```

The production-ready static files are generated in `dist/`.

## Hosting Notes

This is a static site. Any hosting provider that supports static HTML, CSS, and JavaScript can serve the built `dist/` folder.

Common deployment settings:

- Build command: `npm run build`
- Publish/output directory: `dist`
