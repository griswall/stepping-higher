# Stepping Higher Handoff

This package contains the source files for the Stepping Higher coaching website.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build outputs to `dist/`.

## Notes

- Main app code: `src/App.tsx`
- Main styling: `src/App.css`
- Images: `public/images/`
- Build tool: Vite
